# DigiTools

## 📖 Project Description
DigiTools is a modern, responsive web application designed to showcase and manage premium digital products and assets. Built with performance and user experience in mind, it allows users to browse a curated catalog of tools, manage a shopping cart, and complete purchases seamlessly. The project features a clean UI, interactive components, and real-time feedback for all user actions.

---

## 🛠 Technologies Used

- **React.js + Vite**: For fast development and optimized production builds
- **Tailwind CSS & DaisyUI**: For utility-first styling and pre-built UI components
- **JavaScript (ES6+)**: For modern, clean logic and state management
- **React-Toastify**: For elegant user notifications (add to cart, remove, checkout)
- **JSON**: For structured product data management

---

## ✨ Key Features

### 1. 🛒 Dynamic Cart Functionality
Users can add products to their cart with instant feedback. The system supports removing individual items, calculating real-time totals, and clearing the cart upon checkout. The navbar cart icon updates automatically to reflect the current item count.

---

### 2. 📱 Fully Responsive Design
The application is built mobile-first, ensuring a flawless experience across all devices. From navigation menus that adapt to screen size (hamburger menu on mobile) to product grids that shift from 3 columns to 1 column, every element is optimized for Mobile, Tablet, and Desktop views.

---

### 3. 🔔 Interactive User Feedback
Enhanced user experience through React-Toastify notifications. Users receive immediate visual alerts when adding items to the cart, removing items, or completing a checkout. Additionally, product buttons change state (e.g., **"Buy Now" → "Added to Cart"**) to prevent duplicate additions.
